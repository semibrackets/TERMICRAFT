# TERMICRAFT, the topdown terminal minecraft clone

version: 0.0.0-alpha-dev

TERMICRAFT is a sandbox game that takes place in the terminal
You the player '@' can move around freely with wasd and can build with the period and comma keys

CONTROLS:
WASD: move around
COMMA: Mine blocks
PERIOD: Place selected block

ARROW UP: move selected block up
ARROW DOWN: move selected block down

BLOCKS:
Air [ ], 
Grass [.], 
Dirt [%],  
Stone [#],

Note that ncurses doesnt have a brown color for dirt, so i used yellow as a substitude. This will be changed later on.

Thats about it
